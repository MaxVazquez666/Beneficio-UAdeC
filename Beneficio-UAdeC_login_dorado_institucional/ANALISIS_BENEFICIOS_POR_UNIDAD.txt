CÓMO PASAR UN BENEFICIO PENDIENTE A PUBLICADO:
1. Abre data/beneficios-nuevos-pendientes.json.
2. Copia el objeto del beneficio aprobado.
3. Pégalo dentro del arreglo de data/beneficios.json.
4. Quita las propiedades status y source si quieres dejarlo limpio.
5. Sube el commit a GitHub Pages.

NOTA: En GitHub Pages no se puede guardar desde un formulario directamente porque no hay base de datos ni servidor. Para eso se recomienda Firebase o Google Sheets.
