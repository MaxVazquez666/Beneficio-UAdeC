CÓMO SUBIR ESTA VERSIÓN A GITHUB PAGES

1. Descomprime este ZIP.
2. En GitHub, entra al repositorio Beneficio-UAdeC.
3. Da clic en Add file > Upload files.
4. Sube EL CONTENIDO de la carpeta descomprimida, no la carpeta completa y no el ZIP.

Debe quedar en la raíz así:
- index.html
- styles.css
- script.js
- registro.html
- admin.html
- admin.js
- affiliate-register.js
- assets/
- data/

5. Baja hasta Commit changes.
6. Escribe: Actualización navegación y panel UAdeC.
7. Da clic en Commit changes.
8. Espera de 1 a 3 minutos.
9. Abre la página con un número nuevo al final para evitar caché:
   https://maxvazquez666.github.io/Beneficio-UAdeC/?v=3000

SI NO CAMBIA LA PÁGINA
- Revisa en la pestaña Actions que GitHub Pages tenga palomita verde.
- Abre en modo incógnito.
- Usa otro número en ?v=, por ejemplo ?v=3001.
- Verifica que no hayas subido una carpeta completa dentro del repositorio.

FORMA MÁS FÁCIL SIN BORRAR ARCHIVO POR ARCHIVO
Usa GitHub Desktop o VS Code:
1. Clona el repositorio.
2. Borra el contenido local de la carpeta.
3. Pega el contenido nuevo.
4. Haz Commit.
5. Haz Push.
